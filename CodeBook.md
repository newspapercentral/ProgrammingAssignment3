#CodeBook.md

The script produces three variables: testAndTrainRows, filteredData, and meanData.

## Unfiltered Data: testAndTrainRows
testAndTrainRows is the unfiltered data set that combines the activity and subject ID into the feature data. Each row represents and observation and has the folloing columns. Columns 1:561 are the features provided in the original test and train data sets. Column 562 contains the activity labels for those features, and column 563 contains the subject ID for the observation.

## Independent Data Set: meanData
Finally, the meanData contains the average of each variable for each activity and each subject. There is one row for each subject, and each column contains the mean of of the feature for that subject. The columns are the same as the filteredData set.
